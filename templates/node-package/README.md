# Node Package Template

A modern Node.js package template.

## Structure

```
lib/
├── benches/      # Benchmarks
├── examples/     # Usage examples
├── src/          # Source code
├── test/         # Tests
├── .gitignore
├── README.md
└── package.json
```

## Scripts

- `npm run dev` - Run the package
- `npm run test` - Run tests
- `npm run lint` - Lint code
- `npm run verify` - Run all checks
