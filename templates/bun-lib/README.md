# Bun Package Template

A modern Bun package template following best practices.

## Structure

```
lib/
├── benches/      # Benchmarks
├── examples/     # Usage examples
├── src/          # Source code
├── test/         # Tests
├── .gitignore
├── README.md
├── package.json
└── tsconfig.json
```

## Scripts

- `bun run dev` - Run the package
- `bun run build` - Build the package
- `bun run test` - Run tests
- `bun run lint` - Type check
- `bun run verify` - Run all checks
