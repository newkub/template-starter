# Nuxt Monorepo Template

Monorepo with Nuxt app and shared packages using Turborepo.

## Structure

```
├── apps/
│   └── web/          # Nuxt application
├── packages/
│   └── shared/       # Shared code
├── turbo.json
└── package.json
```

## Scripts

- `bun dev` - Start development
- `bun build` - Build all packages
- `bun lint` - Lint all packages
- `bun test` - Test all packages
